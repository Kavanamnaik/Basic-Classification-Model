# Basic Classification Model Using Iris Dataset

## Project Overview

This project is a machine learning classification model developed using the Iris dataset from Scikit-learn. The objective is to classify Iris flowers into three different species based on their sepal and petal measurements. The project also compares the performance of Logistic Regression and Decision Tree algorithms.

---

## Objectives

- Load and explore the Iris dataset.
- Perform Exploratory Data Analysis (EDA).
- Split the dataset into training and testing sets.
- Train classification models.
- Compare the performance of Logistic Regression and Decision Tree.
- Evaluate the models using different performance metrics.

---

## Dataset

**Dataset Name:** Iris Dataset

**Source:** Scikit-learn

**Number of Samples:** 150

**Features:**
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

**Target Classes:**
- Setosa
- Versicolor
- Virginica

---

## Algorithms Used

### Logistic Regression

Used to classify flowers into different species based on feature values.

### Decision Tree Classifier

Used to compare performance with Logistic Regression.

## Results

Both algorithms achieved high accuracy on the Iris dataset. Logistic Regression performed slightly better and was selected as the best-performing model.
